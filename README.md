# -PENJELASAN PROGRAM-
1. Masukkan bilangan sebanyak 3 buah bilangan.
2. Setelah di input, program akan membandingkan ke 3 buah bilangan tersebut.
3. Proses perbandingan menggunakan statement if,elif,else
4. Ketika bilangan pertama (bil_1),dibandingkan dengan bilangan ke dua (bil_2), dan bilangan ketiga (bil_3).
Apabila menghasilkan nilai true, maka akan menghasilkan nilai (bil_1)
5. Bandingkan kembali (bil_2) dengan (bil_1) dan (bil_3). Jika bernilai true maka akan tercetak nilai 9bil_2).
6. Jika perbandingan ke dua tersebut bernilai false, maka akan tercetak nilai (bil_3).
7. Nilai yang tercetak merupakan nilai terbesarna. berikut hasilnya
